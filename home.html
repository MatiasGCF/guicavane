What is Guicavane?
Guicavane it's a program to watch shows and movies provided by sites such as cuevana.tv. It has a nice simple and clean interface that allows fast access to watch shows and movies.
Guicavane lets you choose the player to be used so you don't have to use the flash player used in this websites. You can configure guicavane to use VLC or Mplayer with just a few clicks.
Where can i use Guicavane?
Guicavane it's avaliable for GNU/Linux, Mac and Windows.
Features
  • Streaming shows and movies. Obviously
  • Download them to your hard drive.
  • List of favorites shows.
  • Customizable player.
  • Customizable file name. You can choose the format of the file name to be downloaded
  • Automatic Marks. After watching a show, it marks it so you know exacly witch episodes you have watched
  • Premium Accounts. currenly only Megaupload
  • Multiple Download hosts. You can choose with diferent sources like Megaupload, Hotfile, Filefactory, etc
  • Resume downloads. If you accidentaly close the program, the next time it will start the download right where it left it
